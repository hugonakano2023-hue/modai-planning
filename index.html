Opções — clique para selecionar

A — APIs de Afiliados

Usar APIs da Lomadee, Awin, Amazon Associados, Shopee Affiliates para puxar produtos com link de comissão.

✓ Legal e autorizado
✓ Link de afiliado já embutido
✓ Dados estruturados
✗ Nem todas as lojas têm API
✗ Catálogo pode ser limitado
💰 Custo: R$0 (APIs gratuitas)
B — Web Scraping

Crawlers que visitam os sites das lojas e extraem produtos, preços e promoções em tempo real.

✓ Dados completos e em tempo real
✓ Qualquer loja, sem depender de API
✗ Zona cinza legal
✗ Lojas podem bloquear
✗ Manutenção constante (sites mudam)
💰 Custo: R$100-500/mês (proxies + infra)
C — Curadoria Manual (MVP)

Equipe seleciona manualmente as melhores ofertas/peças todo dia com links de afiliado.

✓ Rápido de implementar
✓ Controle total da qualidade
✓ Zero complexidade técnica
✗ Não escala
✗ Depende de trabalho manual diário
💰 Custo: tempo humano
D — Híbrido (Recomendado MVP)

APIs de afiliados para lojas que têm + curadoria manual para completar. Evolui para scraping depois.

✓ Melhor dos dois mundos
✓ Lança rápido com qualidade
✓ Escala gradualmente
✗ Ainda precisa de trabalho manual no início
💰 Custo: R$0 + tempo humano

Detalhe UX: Animação fluida tipo Tinder. Cada card mostra a foto + tags do estilo. Feedback visual imediato.

US$0,075 por imagem. Com 1.000 usuários fazendo 5 try-ons/mês = 5.000 imagens = US$375/mês (~R$1.900).

Rate limiting é essencial para não estourar o orçamento. O feature gating (free=1, básico=5, premium=ilimitado) protege o custo.
